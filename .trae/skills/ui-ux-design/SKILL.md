# UI/UX Design Skill

## Project Design System

### Color Palette
- **Primary**: `#e74c3c` (Red - for actions, buttons)
- **Secondary**: `#3498db` (Blue - for info, links)
- **Success**: `#27ae60` (Green - for success states)
- **Warning**: `#f39c12` (Orange - for warnings)
- **Background**: `#f5f5f5` (Light gray)
- **Card Background**: `#ffffff` (White)
- **Text Primary**: `#2c3e50` (Dark blue-gray)
- **Text Secondary**: `#7f8c8d` (Gray)

### Typography
- **Font Family**: System default (Chinese supported)
- **Headings**: Bold, dark text
- **Body**: Regular weight, readable size

### Layout Guidelines
1. **Card-based Design**: Use white cards with subtle shadows
2. **Consistent Spacing**: 16px/24px standard padding
3. **Responsive**: Support desktop and tablet views
4. **Clear Hierarchy**: Visual distinction between sections

### Common Components

#### Buttons
- Primary: Red background, white text, rounded corners
- Secondary: White/gray background, dark text
- Danger: Red outline or solid red

#### Forms
- Input fields: Light border, focus state with primary color
- Labels: Clear, above or beside inputs
- Validation: Inline error messages in red

#### Tables
- Striped rows for readability
- Hover effects on rows
- Action buttons in dedicated column

### UX Best Practices
1. **Clear Feedback**: Show loading states, success/error messages
2. **Progressive Disclosure**: Hide advanced options behind toggles
3. **Consistent Actions**: Same action buttons in same positions
4. **Accessibility**: Sufficient contrast, readable fonts
5. **Chinese Language**: All UI text in Simplified Chinese

### AI Search Interface
- Prominent search input with clear placeholder
- Settings gear icon for configuration
- Diagnostic info panel showing parsed results
- Clear indication of filter logic (AND/OR/AT_LEAST)

### Filter Panel
- Collapsible sections
- Clear labels in Chinese
- Dropdown selectors for ranges
- Certificate selection with autocomplete
